# ❤️ Heart Attack Risk Prediction API (Flask)

Machine Learning API untuk prediksi risiko serangan jantung menggunakan Random Forest.

## 🚀 Features

- Load trained ML model
- Scaling with MinMaxScaler
- Risk probability output
- Top 5 feature importance
- Health check endpoint

## 📌 Endpoints

### Health Check

GET /health

### Predict Heart Attack Risk

POST /predict-heart-attack

## 📦 Requirements

Python 3.12+

Install dependencies:

pip install -r requirements.txt

## ▶️ Run Locally

python app.py

Server will run on:
http://localhost:5000
